How to play this game?

This game is played with your mouse buttons or the Z and X keys (they can be changed) Circles and sliders appear on the screen, and you have to hit them to score points. For the most amount of points you need to press the button when the outer circle hits the inner circle.

Cursor movement seems to be delayed?

Try enabling hardware cursors in settings, this can reduce the latency of cursor feedback. However, in some operating systems or browsers, there may be problems such as flickering or the inability to display the cursor.

The game is very laggy, how can i fix this?

You can turn off "Use system resolution" in the settings and set the resolution to a lower value It is recommended to avoid high CPU usage, close antivirus software like Norton 360. it is also recomended to use a modern browser like Fire fox or a Chromeium based browser. if you still encounter problems please report it to Github
